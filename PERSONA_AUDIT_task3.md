# 和訳トーン監査レポート

## 背景

ユーザーから「和訳が馴れ馴れしい」という指摘。以下の前提を踏まえて4件の和訳を検証する。

### 和訳の目的の明確化（最重要）

**和訳は「ユーザーが実際にこう言った日本語」ではない。**

和訳の目的は：
- ユーザーが英文で送った（または送る予定の）メッセージの**意味内容**を日本語で確認できるようにすること
- ユーザーが「自分が何を伝えたか」を振り返れること

つまり和訳は**翻訳的要約**であり、ユーザーの日本語の話し方をシミュレーションするものではない。

### ユーザー像

- 30代男性
- 知的で落ち着いた印象
- 英語メッセージ自体はカジュアル（haha, baby, wanna等）だが、それは**英語のコミュニケーション戦略**であり、日本語の人格とは別
- 日本語ではClaude Codeに対しても丁寧語寄りのニュートラルな口調

### 【Vita 手動送信 1】

**現行訳:**
> セルフィーとかあんま撮らないんだよね、なんか気恥ずかしくて😅
> でもそう言ってくれるの普通に嬉しい🥺 マジで。俺もお前のこと仲いい友達だと思ってるよ、話しやすいし

**問題箇所:**
| 表現 | 問題 | 深刻度 |
|------|------|--------|
| 「お前」(2回) | 相手を「お前」と呼ぶのは乱暴。英語の"you"にはこのニュアンスはない | **致命的** |
| 「俺」 | 英語原文の一人称にヤンキー的ニュアンスはない。ユーザーの日本語人格と不一致 | 高 |
| 「マジで」 | やや砕けすぎ | 中 |
| 「てかちょっと気になったんだけど」 | 10代〜20代前半の口調。30代男性に不適 | 中 |
| 「あんま」「仲いい」 | くだけた省略形。翻訳的要約としては不適切 | 低 |

**修正案:**
> セルフィーはあまり撮らないんだよね、ちょっと気恥ずかしくて😅
> でもそう言ってもらえるのは素直に嬉しい🥺 本当に。自分も君のことは気の合う友達だと思ってるよ、話しやすいし。
> ところでちょっと聞きたいんだけど…君にもそういう人いる？彼氏とか、なんでも話せる相手みたいな。

## 総合分析

### 問題の本質

現行の和訳は**英語メッセージのカジュアルさを「日本語のヤンキー口調」に変換してしまっている**。

英語での"you"は中立的で、友達にも恋人にも使う。これを「お前」と訳すのは、英語にはない粗暴さ・上下関係を持ち込んでいる。同様に、英語のカジュアルな文体（wanna, haha）は自然な口語であり、日本語の「てか」「マジ」「あんま」とは質が異なる。

### トーンガイドライン（修正基準）

| 項目 | NG | OK |
|------|----|----|
| 一人称 | 俺 | 自分、僕、省略 |
| 二人称 | お前 | 君、名前、省略 |
| 接続詞 | てか、つーか | ところで、それと |
| 強調 | マジで、ガチで | 本当に、正直 |
| 省略 | あんま、やっぱ | あまり、やっぱり |
| 文末 | 〜だろ、〜じゃね | 〜だよね、〜かな |

### 目指すべきトーン

**「30代知的男性のナチュラルな口語」**

- フォーマルすぎない（ですます調にはしない）
- 粗暴でもない（「俺」「お前」は使わない）
- 落ち着きがある（「マジで」「てか」は控える）
- 自然体（硬すぎず、読んで違和感がない）

具体的イメージ：友人と落ち着いたバーで話すときの口調。

### 翻訳設定への反映提案

`laura_translation.md` に以下のルールを追加することを推奨：

```
### 和訳（英→日）のトーンルール
- 和訳はユーザーが内容を確認するための翻訳。ユーザーの「日本語での話し方」を再現するものではない
- 一人称: 「自分」または省略。「俺」は使わない
- 二人称: 「君」または相手の名前。「お前」は絶対禁止
- 全体トーン: 30代男性のナチュラルな口語（落ち着き+カジュアル）
- 過度な若者言葉（てか、マジで、ガチ等）は避ける
- 省略形は最小限に（あんま→あまり、やっぱ→やっぱり）
```
