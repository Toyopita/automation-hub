# Windows PC スペック概要

| 項目 | スペック | 評価 |
|:------|:---------|:------:|
| **CPU** | Intel Core i7-6500U (第6世代 / 2015年) | 古い |
| **メモリ** | 8GB | 最低限 |
| **ストレージ** | 500GB HDD (5400rpm) | 遅い |
| **GPU** | Intel HD Graphics 520 (内蔵) | 最低限 |
| **Windows 11** | 非対応 (第8世代以降のCPUが必要) | - |
| **Windows 10** | サポート終了済 (2025年10月) | 注意 |

---

# 向いている用途

| 用途 | 評価 | 備考 |
|:------|:------:|:------|
| プログラミング学習 (Python, HTML/CSS) | **◎** | VSCode + 軽量言語OK |
| Office操作の練習 | **◎** | Word / Excel / PowerPoint |
| タイピング練習 | **◎** | 問題なし |
| ブラウザでの調べ物 | **○** | タブ10個程度まで |
| Git / GitHub 学習 | **○** | CLI操作なら余裕 |
| Linuxを入れて勉強 | **◎** | Ubuntuなら軽快に動く |

---

# 向いていない用途

| 用途 | 評価 | 理由 |
|:------|:------:|:------|
| 実務（複数アプリ同時） | **×** | メモリ不足でフリーズ |
| Web会議 + 作業 | **×** | CPU 2コアでカクつく |
| 動画編集 / ゲーム | **×** | GPU・CPU共にスペック不足 |
| 大量データ処理 | **×** | HDD + メモリ8GBで限界 |

---

# まとめ

## 練習・学習用PCとしてはまだ使えます

- 実務利用は難しい
- プログラミング入門・Office練習に最適
- SSD換装 (約3,000円) すれば体感が大幅改善
- Windows 10はサポート終了済 → ネット接続はセキュリティリスクあり
