# Marp サンプルスライド

Claude が作成したスライドです

**作成日**: 2025年11月11日

---

## Marpとは？

- **Markdown** でスライドを作成できるツール
- テキストエディタで編集可能
- Git でバージョン管理が容易
- 複数の形式にエクスポート可能
  - HTML
  - PDF
  - PowerPoint (PPTX)

---

## 使い方

スライドの区切りは `---` を使います。

```markdown
---
marp: true
---

# 最初のスライド

内容

---

# 次のスライド
```
